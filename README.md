# Library Management System

A console-based library management app built in Dart, covering core language features, OOP, async programming, and concurrency.


Challenge Link : https://dart-challenge.netlify.app/


## Run

```bash
dart bin/library_management.dart
```

> Requires [Dart SDK](https://dart.dev/get-dart) 

---

## Features

| Feature | Description |
|--------|-------------|
| Books | Add, list, search books with genre tags |
| Members | Register standard & premium members |
| Loans | Loan and return books with full history |
| Reservations | Premium members can reserve available books |
| Isolate Search | Full-text catalog search runs off the main thread |
| Live Log | Every action broadcasts to a real-time activity stream |
